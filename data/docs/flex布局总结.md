# Flex 布局总结

## 核心概念
- `display: flex` 会把容器变成弹性盒子。
- 主轴由 `flex-direction` 决定，默认是水平。
- 交叉轴与主轴垂直。

## 常用属性
- `justify-content` 控制主轴对齐。
- `align-items` 控制交叉轴对齐。
- `flex-wrap` 控制是否换行。
- `gap` 用于控制项目间距。

## 实战经验
1. 先确定主轴，再决定对齐方式。
2. 复杂布局先用容器拆分，再用子容器微调。
3. 常见页面：左右布局、居中布局、卡片流都可以用 Flex 解决。

## 面试回答模板
Flex 的本质是“沿着主轴分配空间”。当我需要快速做响应式布局时，通常会先用 Flex 做骨架，再配合 `gap`、`flex: 1` 和 `align-items` 调整细节。
